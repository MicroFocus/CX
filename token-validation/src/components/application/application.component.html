CX and OSP Token Validation

This application shows you how to use the OSP integration service in the CX middle tier.

With a valid token we can get information about the current user like the username: {{$ctrl.tokenInfo.username}}.

We can also make an extra rest call to get additional information about the user like:

Given name {{$ctrl.userAtts.first_name}}
Family name {{$ctrl.userAtts.last_name}}
DN {{$ctrl.userAtts.name}}

You can also logout and try a different user.
